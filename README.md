### Key Features
-   **Auto-Start Proxy**: Never run `npm start` manually again.
-   **Real-Time Usage**: Monitor quota limits for Gemini 3 Pro/Flash and Claude.
-   **Multi-Account**: Automatic load balancing and failover.
-   **Status Bar**: Quick access via `$(server) Claude Proxy`.
